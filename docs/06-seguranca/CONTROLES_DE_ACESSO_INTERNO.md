# Controles de Acesso Interno

- RBAC: Owner, Manager, Accountant, Employee; permissões por recurso (vide RBAC.md).
- Segredos: variáveis em .env/.env.local, chaves JWT em arquivos protegidos; usar secrets em CI/CD.
- Acesso a DB (Neon): usuários separados por ambiente; rotação de credenciais.
