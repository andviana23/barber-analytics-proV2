# Fluxos de Negócio

- Financeiro: lançamento de receitas/despesas → fluxo de caixa → dashboards.
- Assinaturas: criação/edição → Asaas → faturas → repasse.
- Lista da Vez: entrada de barbeiro → atendimento → histórico/pontos.
- Onboarding: signup → onboarding wizard → habilitação de tenant.
